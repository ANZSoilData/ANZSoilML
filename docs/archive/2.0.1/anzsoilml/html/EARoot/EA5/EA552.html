The ANZSoilML GlobalSoilMap application schema is an application of GML/Observations and Measurements that has been developed by members of the Oceania node of the GlobalSoilMap.net project (http://globalsoilmap.net/). It uses the ANZSoilML extensions of O&M to create GML feature types that provide meta-data about:<br /></p><p>- GSM spatial entities as a special type of ANZSoilML SoilProfile (an O&M SF_SpatialSamplingFeature)<br /></p><p>- The depth intervals as a special type of ANZSoilML SoilLayer (an O&M SF_SpatialSamplingFeature)<br /></p><p>- The predicted values as an om:OM_Measure.<br /></p><p><br /></p><p>This application schema conforms to version 2.3 of the GlobalSoilMap.net Specifications (http://globalsoilmap.net/specifications).<br /></p><p><br /></p><p>Copyright (c) Landcare Research (NZ) Ltd 2015. All rights reserved.<br /></p>
