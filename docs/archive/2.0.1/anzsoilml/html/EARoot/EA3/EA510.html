Notes: The term used to classify the SoilProfile. #lt;br /#gt;#lt;/p#gt;#lt;p#gt;ASLS applies this to a SoilProfile using Australian Soil Classification (Isbell 2008). #lt;br /#gt;#lt;/p#gt;#lt;p#gt;SoilsProfiles may be classified according to a number of schemes. As such the classifier is a DataType that allows using multiple schemes, specified via a citation, for the classification.  #lt;br /#gt;#lt;/p#gt;#lt;p#gt;A (future) alternative is to use a controlled vocabulary that allows specifying at least the scheme name, value name and description.#lt;br /#gt;#lt;/p#gt;#lt;p#gt;#lt;br /#gt;#lt;/p#gt;#lt;p#gt;The classifier should be appropriate to the soil feature being described, whether Soil, SoilProfile or Horizon.#lt;br /#gt;#lt;/p#gt;#lt;p#gt;#lt;br /#gt;#lt;/p#gt;#lt;p#gt;For an Horizon example the ASLS Field Handbook contains three horizon classification schemes (these may logically all fall into a single scheme): #lt;br /#gt;#lt;/p#gt;#lt;p#gt;ASLS Field Handbook 'Master' (A1, B2, ...) (p148)#lt;br /#gt;#lt;/p#gt;#lt;p#gt;ASLS Field Handbook 'SubDivision' (1,2,3, ...) (p155)#lt;br /#gt;#lt;/p#gt;#lt;p#gt;ASLS Field Handbook 'Suffix' (b, c, d, e, ...) (p153)#lt;br /#gt;#lt;/p#gt;#lt;p#gt;#lt;br /#gt;#lt;/p#gt;#lt;p#gt;An alternative scheme is the classification of a horizon according to the FAO classification system. A code system representing horizons where the same dominant soil forming processes have been active. This code summarizes many observations of the soil description and gives an impression about the genetic processes that have formed the soil under observation. #lt;br /#gt;#lt;/p#gt;#lt;p#gt;NOTE The horizon notation is a combination of several symbols.#lt;br /#gt;#lt;/p#gt;#lt;p#gt;SOURCE Guidelines for Soil description. FAO, 2006#lt;br /#gt;#lt;/p#gt;#lt;p#gt;EXAMPLE Bw: meaning B horizon with a development of colour and/or structure.#lt;br /#gt;#lt;/p#gt;
