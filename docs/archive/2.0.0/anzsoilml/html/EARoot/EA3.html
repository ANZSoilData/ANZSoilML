Summary schema to import all related application schemas for ANZSoilML v1.0.0<br /></p><p>ANZSoilML is an application schema that specifies a set of feature-types and supporting structures for information used in the Australian and New Zealand soil sciences.<br /></p><p>ANZSoilML v2 is scoped to the information specified in:<br /></p><p>    AUS: Australian Soil and Land Survey Field Handbook, Third edition, 2009, The National Committee on Soil and Terrain.<br /></p><p>    NZL: Soil Description, Revised edition, 1995, Milne, J.D.G., Clayden, B., Singleton, P.L., Wilson, A.D.<br /></p><p>ANZSoilML v2 is primarily concerned with observed properties of Soils and associated Landscapes.<br /></p><p>ANZSoilML v2 is factored into a set of sub-packages that are maintained separately. <br /></p><p>---<br /></p><p>ANZSoilML v2 improves the concepts and properties in ANZSoilML v1, which was developed under the auspices of the Australian Collaborative Land Evaluation Program (ACLEP) http://www.clw.csiro.au/aclep/, and Landcare Research, New Zealand.<br /></p><p>Copyright (c) CSIRO and LCR 2014.
