The geometry of SoilSample should be a 0D point, 1D profile or 2D polygon, not 3D volumes. Therefore the depthRange of the sample is specified rather than being included in the geometry property. Where a 3D geometry is provided the depthRange will duplicate the 3D geometry depth.
