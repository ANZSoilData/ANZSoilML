A SoilHorizon is a layer of soil, approximately parallel to the land surface, with mostly homogenous morphological properties different from layers below and/or above it. (p148)<br /></p><p>Distinguished from SoilLayers in that it is developed in a parent material layer through pedogenic processes or made up of in-situ sedimented organic residues of up-growing plants (peat) (INSPIRE), and therefore can be classified with a horizon notation.<br /></p>
